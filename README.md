## How to push your work to GitHub

Whenever you are ready to save and upload your changes to GitHub, you can run the following commands in your terminal:

1. **Stage your changes:**
   ```bash
   git add .
   ```

2. **Commit your changes:**
   ```bash
   git commit -m "Add bootcamp work"
   ```

3. **Push to GitHub:**
   ```bash
   git push -u origin main
   ```
